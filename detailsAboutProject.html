Steps to Create Your Own Library and NPM PACKAGES

Step1

note:- use this command to create a WORK SPACE, create application = false doesn't create src outside project and other project things just create work space, recommended to use for library

ng new magic-strings-exapmple --create-appllication= false.

Step2) create library using this ,add prefix will add prefix while cretaing component

ng generate libarary library-name --prefix=ak

Step 3) run project using ng serve proj name

ng serve proj-name or ng serve --open

Step4) it will create ur library in dist folder

ng build library-name

To use library in project

Step1 Go to app module add library exapmle in this project u found magicStringModule

Step1 use selector to use this library for example

note :- each time we make any change in library we have to use "ng build library-name" for convinent in this project go to package.json ,u will fing "lib:build" to run same command. now just write "npm run lib:build" and your library build.

To CONVERT IN PACKAGE

  • cd dist
  • cd magic-string
  • npm pack
or JUST add " "lib:package" : "cd dist && cd magic-string && npm pack" " in package.json and write npm run lib:package
